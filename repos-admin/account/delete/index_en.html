Delete account

This has to be done manually. Repos can not know how the user has been integrated with the repository. It usually involved these steps:

  1. Remove the [/username] section from the ACL.
  2. Remove the username from any groups that he/she is member of.
  3. If the user has a home folder, that folder can be deleted. This does not free up any storage space, but makes the folder name available for others. Please note that if others are given access to that path in the ACL, they can also access the history.
  4. The user may still have an account stored in the Repos server for some time, but if there are no entries in the ACL he/she can not access repository contents. If this is a problem, ask a server administrator to remove the account from the users file.

This process can of course be automated to suit you. Please contact the Repos team for more information.
