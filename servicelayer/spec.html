Specification: Subversion Service Layer

svn=log

  • text/xml
  • The log for the URL as XML
  •  

rev=X

  • unknown/unknown
  • Get the contents of a specific revision
  •  

svn=list

  • text/xml
  • The XML output from svn list command on the folder or file
  •  

svn=listjs

  • text/javascript
  • A javascript that writes a folder contents to an HTML element
  • (no selector)Without GET parameters, this service returns the svn list as an object "svn" in JSON.
  • selector CSS selector for the UL element to add contents to. If the selector is a string of a-z0-9_-, an '#' will be prepended by the service (because #id is the most common selector, and URLs with brackets are tricky)
  •  
  • The page must import jQuery before using this script with a selector

POST

  • text/plain
  • An HTTP post request to a repository file url overwrites the file with the posted contents
  • usertext The posted plaintext contents for a new version
  •  
  • Subversion does not respond to post, so without the service layer a 404 error is returned
