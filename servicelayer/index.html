Subversion Service Layer

Use your repository as Web Service or AJAX data. Integrate your organization's data with your systems and websites. The Subversion Services are called by appending simple queries to your URLs for documents and files.

  • Save your data in a file structure instead of a database
  • Access any version of a document using a web browser
  • Easily add lists of files to intranet pages
  • Post information from any system directly to your repository
