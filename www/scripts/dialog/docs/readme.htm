Dialogs - Floating windows

Browsers

Confirmed to work in Windows XP with IE, Fireworks, and Opera

Requirements

prototype.js library
scriptaculous.js library

Please put the following into the head section of your html file: 

<script type="text/javascript" language="javascript" src="js/scriptaculous/lib/prototype.js"></script>
<script type="text/javascript" language="JavaScript" src="js/scriptaculous/src/scriptaculous.js"></script>
<script type="text/javascript" language="JavaScript" src="js/extensions/_dialog.js"></script>

Creating a new dialog

var dlg0 = new Dialog(null, {options});
var dlg1 = new Dialog(null, {options});

Convert existing DIV(s) to dialog(s)

var dlg = new Dialog(element, {options});

or

var els = document.getElementsByClassName('dContainer');
var dlg = Array();
for (var i = 0; i < els.length; i++) {
	dlg[i] = new Dialog(els[i], { resize: true, drag: true, width: 200, height: 200, modal: false });
}

Remark: Please use html markup in /template/dTemplate.html

Options
title title: 'str' - sets the dialog title
top top: 10 - sets the distance form the top of the window in pixels
left left: 10 - sets the distance from the left of the window in pixels
width width: 400 - sets the with of the dialog in pixels
height height: 300 - sets the height of the dialog in pixels
mwidth mwidth:80 - sets the minimum width of the dialog in pixels when resizing
mheight mwidth:30 - sets the minimum width of the dialog in pixels when resizing
moveable movable: true - true/false to allow moving of dialog; default: true;
resizable resizable: true - true/false to allow resizing of dialog; default: true;
minimizable minimizable: true - true/false to allow minimizing of dialog; default: true;
maximizable maximizable: true - true/false to allow maximizing of dialog; default: true;
closeable closeable: true - true/false to allow closing of dialog; default: true;
scrollbar scrollbar: 'hidden' - 'auto', 'scroll', 'hidden'; default: hidden;
statusbar statusbar: true; true/false to enable statusbar - if set to false - resizing is disabled as well
modal modal: true; true/false to show dialog in modal mode; default: false;
show show: true; true/false to show dialog on init - if set to true, dialog will be positioned automitically
remove remove: true; true/false to remove dialog from DOM on closing dialog; default = true
effects effects: true; true/false to use dialog transition effects; default = true
limit limit: true; true/false to limit dragging and moving of dialog to main container (client window)
debug debug: true; true/false to show debug messages in status bar - if set to true, width and height are displayed upon resizing of dialog.

Manipulating a dialog (methods)

Manipulating
show(top) this function shows the dialog. The 'top' argument is optional - if set to true, the dialog will be raised to the top.
open(top) this function is an alias to show()
remove() this function closes the dialog and remove it from the DOM; all event observers are removed as well; re-opening dialog not possible
close() this function closes the dialog; allows re-opening of dialog
hide() this function is an alias to close()
top() this function raises the dialog to the top.
move(x, y, mode) this function moves the dialog to the x, y coordinates, e.g. (0,0) = top left corner. The mode argument is optional and can be set to either true or false - default = true. If mode is set to false, no transition is used when moving the dialog.
position(x, y) this function moves the dialog to the x, y coordinates, e.g. (0,0) = top left corner. This function is basically an alias to move().
center() this function centers a dialog both vertically and horizontally
centerX() this function centers a dialog horizontally - current vertical position will be kept unchanged
centerY() this function centers a dialog vertically - current horizontal position will be kept unchanged
resize(width, height, mode) this function resizes the dialog. The mode argument is optional and can be set to either true or false - default = true. If mode is set to false, no transition is used when resizing the dialog.
maximize() this function resizes the dialog to the size of the screen size.
minimize() this function hides the content area and the status bar. Dialog title remains visible.
visible() this function returns true if dialog is shown, false if not
setTitle(str) this function sets the title of a dialog to str
clearTitle() this function clears the title of a dialog
addContent(str) this function adds content to the dialog
setContent(str) this function adds content to the dialog - alias to setContent()
clearContent() this function clears the content of a dialog
prependContent(str) this function prepends str to the content
appendContent(str) this function appends str to the content
setStatus(str, fade) this function sets the text of the status bar to str; if the fade argument is set to 'true', the status text will be faded after it has been shown
clearStatus(fade) this function clears the text of the status bar; if the fade argument is set to true, the status text will be faded to clear
scrollbar(scroll) this function sets the scrollbar of the dialog; possible values are: 'auto', 'scroll', 'hidden'; default: 'hidden'
getTheme() this function returns the current active theme of a dialog
setTheme(str) this function sets the theme of a dialog to 'str'
