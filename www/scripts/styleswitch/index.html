Javascript stylesheet switcher

Select a theme by clicking the link:

  • Repos - this is the default
  • Project Engine - used for the integeration with Project Engine

When selection is done: proceed to login.

If the menu images flicker in Internet Explorer, your browser needs this fix from Microsoft. Simply paste javascript:void(document.execCommand("BackgroundImageCache",false,true)) into the address bar.

In Firefox, you can select style any time using the Page style menu.
