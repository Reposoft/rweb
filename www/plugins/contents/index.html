Table of contents

This plugin automatically creates a table of contents.

Background

All word proccessing software have such a feature. And unlike many other features, it is appreciated.

Motivation

Why do we think it is worth the complexity?

No manual labor

We want a simple online word processor with HTML as document format. And users don't want to add the numbering manually.

Good for navigation too

With a clickable table of contents, we enable quick navigation in the document.

Design principles

Like all Repos javascripts, this is non intrusive. It is a behaviour that can be dynamically added to the page.
