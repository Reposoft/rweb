Recommended Subversion server for Windows

Requirements:

  • Support Windows 2000, XP and 2003 server
  • Run in parallell with Internet Information Server
  • Apache 2.0, because it is required by Subversion (2.2 is currently not supported)
  • OpenSSL, to support Basic HTTP authentication
  • PHP 4.3+, to be able to add web interface
  • Subversion 1.4+

All components are open source. They are standard in most Linux dsitributions, but tricky to install in Windows. When bought for self-hosting, Repos is delivered with a preconfigured server installation that meets these requirements.

Running Project Engine version control without Repos

With a Project Engine enterprise license, version control support is included. To create a repository hosted on a Windows server, the Repos server installation can save a lot of time. When the repository is up and running, the Repos web application can be added at any time.
