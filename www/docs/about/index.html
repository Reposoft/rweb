Här är användarkontot "svensson" skrivskyddat, för att kunna vara öppet för alla. I filarkivet finns bland annat dokumentation.

För att testa att ändra fritt i repos.se, logga in på test.repos.se. Det kontot nollställs regelbundet, så där kan alla husera fritt.

Vad behövs? En webbläsare. Vi rekommenderar Firefox. Allt funkar på Internet Explorer också. Vi testar även med Opera, Camino och Safari, men prioriterar inte fullt stöd för dem.
Certifikat

För att använda en säker anslutning, vilket behövs så fort man skickar lösenord över internet, krävs ett certifikat. Repos använder certifikatet http://www.repos.se/CA.crt

Som standard måste certifikatet godkännas innan man loggar in (det kommer automatiskt upp en dialogruta där man godkänner). Om certifikatet importeras till webbläsaren behöver det inte godkännas vid varje inloggning.
