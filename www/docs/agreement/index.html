Use of 3rd party tools

Repos is a user interface and an administration tool for Subversion. Repos web technology relies on open source jQuery, Smarty, Services_JSON, TinyMCE and dp.SyntaxHighlighter. The repository browser includes file icons from the Cezanne Icon Theme with Creative commons license. Automated system test is done with the help of Selenium, Simpletest and ECMAUnit.

Please review the 3rd party tool selection at the lib folder. The administrator has accepted the licenses of installed tools.

All of these tools are used in original released form, configured but not modified for Repos.
