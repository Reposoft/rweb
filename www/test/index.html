Integration Testing

This is the startpage for automated and manual tests of repos.

Repos components are as separated as possible. The unit tests for components are usually in the same folder as the code.

return to repos startpage

Automated test suites

  • Browser test for new users
  • Configuration test
  • Automated System Test
  • PHP unit test suite
  • Javascripts unit tests

Note that tests assume the web application to be located at [current host]/repos/.

Some test cases reveal server details. They only modify the test repository, but it could be a good idea to restrict access to this folder to internal hosts.
