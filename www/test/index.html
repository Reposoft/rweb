Repos components are as separated as possible. The unit tests for components are usually in the same folder as the code. See the php unit tests and the javascripts unit tests.

Automated test suite

Complete test suite for Selenium. A global javascript function called setUp(), from the suite HTML file, is executed at the top of each test to get to the startpage with a user logged in. Then relative URLs are used to go directly to the target for each test case.

Basic test suite written in java wih jWebUnit, to test the functionality without javascripts.
