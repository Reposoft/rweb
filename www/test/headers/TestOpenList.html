If the XML data for an IE ajax request does not have correct headers, IE might hang at 100% CPU.
Check the HTTP headers of the list xml service
open /repos/test/headers/
clickAndWait listxml
verifyTextPresent |HTTP/1.1 200 OK|
verifyTextPresent |Content-Type: text/xml; charset=utf-8|
verifyTextPresent |Content-Length: 399|
