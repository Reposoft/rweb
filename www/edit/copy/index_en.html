Makes a copy of a file or folder at a new location in the repository. Optionally the name can be changed too. Optionally delete the original file or folder (which makes a move operation).

Copies inside the repository take no extra storage space. It is the current revision that is copied. So a copy can be a good way to share a file as it looks now, but without sharing the future changes to the original.
