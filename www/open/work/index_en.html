There are different methods for updating a version controlled file.
- If you will be the only one who makes modifications to the file, you can do the simplest thing: download - modify - upload.
- If you think that other users will be updating the file too, make a personal copy (a branch) of the file, modify the file in the branch, then use the "merge" functionality to add your changes to the common file.
- If you simply want to view the file, do download below. Or use the filename links in the repository.

Download

Download to your local hard drive. When work is completed, use "upload new version" to commit.

upload new version

Upload is good for quick changes in a file that you recently downloaded. For big changes, use a personal branch.

Create personal branch

The file you wish to branch has to be in trunk directory

Your personal, temporary, copy would be: /branches/{=$newname}

create branch

You will get a link to the branch, and from there you can access the "merge" functionality.
