Porting the excellent and highly addictive Quicksilver to javascript

User guide

If a page has the logo, drop the mouse go [Alt + Q]. This brings up a box. The upper part of it is input from your keyboard. The lower part is links from the current page, matching that input. Just hit a key to filter the matches. Hit one more to filter more.

To accept the current selection, hit [enter] or [tab].
Change selection with [arrow up] and [arrow down].
Cancel using [esc].

Use this to navigate between pages with keyboard.
