Repos Quay user guide

If a page has the logo, drop the mouse go [Alt + Q]. This brings up a box. The upper part of it is input from your keyboard. The lower part is links from the current page, matching that input. Just type a character to filter the matches. Type one more to filter more.

To accept the current selection, hit [enter] or [tab].
Change selection with [arrow up] and [arrow down].
Cancel using [esc].

Use Repos Quay to navigate between pages using your keyboard. Ideal for web based file browsing, wiki navigation and browsing reference docs.
